# Demostración visual de temas en EQUIPOS 4.0

## Temas Implementados

La aplicación EQUIPOS 4.0 incluye 4 temas modernos que el usuario puede cambiar desde el menú Configuración → Tema:

### 1. Tema Claro (Por defecto)
- Fondo: Gris claro (#F0F0F0)
- Texto: Negro
- Resaltado: Azul (#0078D7)
- Ideal para: Trabajo diurno, máxima legibilidad

### 2. Tema Oscuro
- Fondo: Gris oscuro (#2D2D30)
- Texto: Blanco
- Resaltado: Azul claro (#2A82DA)
- Ideal para: Trabajo nocturno, reduce fatiga visual

### 3. Tema Azul Corporativo
- Fondo: Azul muy claro (#EBF1F7)
- Botones y menús: Azul corporativo (#0078D7)
- Texto: Negro sobre fondos claros, blanco sobre azul
- Ideal para: Apariencia profesional y corporativa

### 4. Tema Morado Moderno
- Fondo: Morado muy claro (#F5F0FA)
- Botones y menús: Morado vibrante (#9333EA)
- Texto: Negro sobre fondos claros, blanco sobre morado
- Ideal para: Apariencia moderna y distintiva

## Características de cada tema

Todos los temas incluyen:
✓ Paletas de colores coherentes
✓ Estados hover y pressed en botones
✓ Menús estilizados
✓ Tabs personalizados
✓ Tooltips visibles
✓ Colores para estados deshabilitados

## Cómo cambiar de tema

1. Abrir la aplicación EQUIPOS 4.0
2. Ir al menú: Configuración → Tema
3. Seleccionar el tema deseado
4. El tema se aplica inmediatamente y se guarda en la configuración

El tema seleccionado persiste entre sesiones.
